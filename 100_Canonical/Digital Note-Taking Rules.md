## These rules are [[AI-First|optimized for AI use]]. Human use is secondary.

### Note contents:
- Note exists to express a single meaning
- Any single note should make sense when read alone
- If a note depends on another idea, it links to it explicitly
### Note title:
- Must be capitalized
- Must represent 1 idea
### Main Directory title:
- Must be in this format: N_Title (N - hierarchy prefix number)
### Links:
- Links are semantics not navigational
- Links exist to express *meaningful relationships*
### Dates:
- Use `YYYY-MM-DD` (2026-02-12)
- Date + time: `YYYY-MM-DDTHH:MM` (2026-02-12T14:30)
- Date + time + timezone: `YYYY-MM-DDTHH:MM:SSZ` (2026-02-12T14:30:45Z)
### Metadata:
- Metadata is *machine truth*
- Any metadata not in Metadata directory inside [[Canonical Note|Canonical directory]] belongs to the [[Legacy Note]]s
