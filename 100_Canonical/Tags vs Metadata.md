### Metadata / Front-matter:
- Use metadata for machine decisions
- Use when the information should **change AI behavior** or be **parsed deterministically**
- Metadata is in singular form
### Tags:
- Use tags for human grouping
- Use when the information is **descriptive, flexible, or contextual**
### Decision rule:
Ask one question: *if an AI ignored this, would something break?*
• Yes → metadata  
• No → tag
